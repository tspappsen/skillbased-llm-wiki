# RAW TRACKING — Maintain `raw/files.log`

## When to Read

Read this reference before any step that checks drift, refreshes `files.log`, or validates raw source references. This is a supporting reference used by INIT, INGEST, and LINT.

## Scope

`raw/files.log` is a plain inventory of the files in `raw/`. No hashing, no fingerprinting, no modified-file detection — just filenames and dates.

Rules:

- Track only top-level files directly inside `raw/`
- Ignore `files.log`, `.gitkeep`, `.DS_Store`, and hidden files whose names start with `.`
- Never hand-edit individual entries; regenerate the whole file on sync

## File Format

Use this exact header:

```text
# Auto-generated by llm-wiki. Do not edit manually.
# filename | last_updated
```

Then one line per tracked file, sorted by filename:

```text
<filename> | <YYYY-MM-DD>
```

Use the current date in `YYYY-MM-DD` format. Do not include a time, timezone, hash, size, or status column.

## Sync Procedure

Use this when the wiki needs `raw/files.log` refreshed.

1. List the top-level files in `raw/`
2. Drop ignored and hidden files
3. Rewrite `raw/files.log` from scratch using the header and sorted entries
4. Use the current date for `last_updated`

## Status Procedure

Use this when the user asks whether `raw/` changed, or when LINT checks for drift.

1. List the current files in `raw/` (applying ignore rules)
2. Read `raw/files.log`
3. Compare filenames
4. Report two categories:
   - `new` — files in `raw/` but missing from `files.log`
   - `deleted` — files in `files.log` but missing from `raw/`
5. Do not write anything during status

If there are no differences, report that no changes were detected.

Use this exact status shape:

```text
Raw drift:
- new: <comma-separated filenames or none>
- deleted: <comma-separated filenames or none>
```

## Reconciliation Rules

- New raw sources are intentional → ingest them, then sync `files.log`
- Deletions are intentional → confirm the current `raw/` state is correct, then sync
- Unexpected drift → report it before changing anything
